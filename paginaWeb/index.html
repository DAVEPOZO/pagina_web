¿Qué es la Programación?

La programación es una disciplina enfocada en la solución de problemas mediante la definición de pasos precisos que permiten obtener una respuesta adecuada. Estos pasos son expresados en un lenguaje comprensible para una computadora, permitiendo la ejecución de tareas específicas.

Un programa puede entenderse como una secuencia de instrucciones diseñadas para modificar un estado inicial, atravesar diferentes estados intermedios y llegar a un estado final que representa la solución esperada del problema.

Para comprender mejor estos conceptos, consideremos algunas definiciones clave:

  • Acción: Es cualquier evento desencadenado por un agente que ocurre en un tiempo finito y genera un resultado específico.
  • Variables de estado: Son valores que describen el estado actual del programa en todo momento.
  • Cambio de estado: Ocurre cuando una acción modifica los valores de las variables de estado, generando una transición desde un estado inicial a uno final.
  • Ambiente: Representa el conjunto de variables u objetos que definen la situación en la que opera el programa. Este ambiente cambia dinámicamente a medida que las acciones se ejecutan, generalmente bajo la forma de instrucciones dadas a la computadora.

Proceso de Resolución de Problemas

No existe un único método universal para resolver problemas mediante programación, por lo que es recomendable seguir un enfoque estructurado. Una estrategia adecuada permite analizar el problema en profundidad, organizar los pasos lógicos necesarios y desarrollar soluciones eficientes.
Este proceso suele incluir:

  1. Análisis del enunciado del problema
  2. Estrategia de solución
    • Método de solución.
    • Descripción del ambiente.
    • Algoritmo.
  3. Síntesis y Codificación.
  4. Puesta en marcha y validación de los resultados.
