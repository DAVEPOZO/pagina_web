DEFINICION

La programación estructurada se fundamenta en el uso de tres tipos de instrucciones para desarrollar un programa informático. Entre sus ventajas destacan la simplicidad y rapidez en la creación de software, así como la reducción de la complejidad en las pruebas y el proceso de testing, gracias a su enfoque basado en un número limitado de estructuras de control.

1️⃣ Variables

Una variable es donde se almacenan y se recuperan los datos de un programa. Así de simple. En programación, la utilizamos para guardar datos y estados, asignar ciertos valores de variables a otras, representar valores de expresiones matemáticas y mostrar valores por pantallas.

Una variable en programación se caracteriza por:

  • Nombre único: Identifica la variable dentro del programa.
  • Valor guardado: Puede cambiar durante la ejecución del programa.
  • Tipo de datos: Puede ser numérico, texto, booleano, etc.
  • Alcance: Puede ser global (accesible desde cualquier parte del programa) o local (limitado a un bloque de código).

Tipos de variables en programación

Los tipos de variables dependen del lenguaje de programación. Cada lenguaje tiene sus propias formas de definir variables y reglas de tipado:

  • Tipado estático: El tipo de variable se establece en la declaración.
  • Tipado dinámico: El tipo de variable se asigna en tiempo de ejecución.

Algunos tipos de datos

  • Variables numéricas: Almacenan números enteros.
  • Variables de punto flotante: Guardan números con decimales.
  • Variables de caracteres: Representan símbolos o caracteres individuales.
  • Variables de cadena de caracteres: Contienen palabras y textos.
  • Variables lógicas o booleanas: Almacenan valores verdadero o falso.

Cómo nombramos las variables

Lo primero que tienes que saber es que las variables se identifican. Tienen un identificador, que no es otra cosa que la forma en que las nombramos. Y hay que respetar una serie de restricciones o condiciones previas.
Aquí van los límites:

  • El nombre de la variable no puede empezar por un número.
  • El nombre de la variable no puede contener espacios.
  • El nombre de la variable no puede contener caracteres especiales.
  • El nombre de la variable no puede ser una palabra reservada del lenguaje.

                
    # Ejemplo en Python
    nombre = "Juan"
    edad = 25
    print(f"Hola, mi nombre es {nombre} y tengo {edad} años.")

2️⃣ Condicionales

Las condicionales en programación son estructuras fundamentales que permiten a un programa tomar decisiones en función de ciertas condiciones. Actúan como puntos de control que determinan qué conjunto de instrucciones debe ejecutarse dependiendo del resultado de una expresión lógica.

En otras palabras, una condicional permite que el código siga diferentes caminos según se cumplan o no ciertas condiciones, lo que hace que los programas sean más dinámicos y adaptables a distintos escenarios.

Estructura básica de las condicionales

En la mayoría de los lenguajes de programación, la estructura de una condicional sigue un patrón de tipo "si-entonces", siendo la sentencia if una de las más utilizadas. Por ejemplo, en Python se escribe de la siguiente manera:

                    
    if condicion:
    # Código a ejecutar si la condición es verdadera
    else:
    # Código a ejecutar si la condición es falsa

Aquí, el programa evalúa la condición establecida. Si esta es verdadera, se ejecuta el bloque de código dentro del if; en caso contrario, se ejecuta el bloque definido en else. Esta estructura permite modificar el flujo del programa de forma sencilla y eficiente.

Operadores de comparación y condiciones combinadas

Las condicionales suelen utilizar operadores de comparación para evaluar expresiones. Algunos operadores comunes incluyen:

  • == (igualdad)
  • != (diferente)
  • > (mayor que)
  • < (menor que)
  • >= (mayor o igual que)
  • <= (menor o igual que)

Además, es posible combinar múltiples condiciones usando operadores lógicos como and (y) y or (o), lo que permite generar reglas más complejas para la toma de decisiones. Veamos un ejemplo:


    # Ejemplo en Python
    edad = 20
    if edad >= 18 and edad <= 65:
        print("Eres un adulto.")
    elif edad < 18:
        print("Eres menor de edad.")
    else:
        print("Eres un adulto mayor.")

En este fragmento de código, la variable edad es evaluada en distintas condiciones. Dependiendo del valor, el programa imprimirá un mensaje diferente, lo que demuestra la utilidad de las condicionales en la toma de decisiones dentro de un programa.

Permiten ejecutar código según una condición.

                
    # Ejemplo en Python
    if edad >= 18:
        print("Eres mayor de edad.")
    else:
        print("Eres menor de edad.")

3️⃣ Bucles

En programación, los ciclos, también conocidos como bucles, son estructuras que permiten ejecutar repetidamente un conjunto de instrucciones mientras se cumpla una condición específica. Esto permite automatizar tareas repetitivas sin necesidad de escribir el mismo código varias veces.

El funcionamiento de los ciclos está controlado por variables o condiciones lógicas, lo que nos permite definir el número de repeticiones y evitar bucles infinitos que podrían bloquear el programa.

Tipos de Bucles en Programación

Los bucles más utilizados en la mayoría de los lenguajes de programación son:

Bucle for

El bucle for es ampliamente utilizado cuando se conoce de antemano la cantidad de veces que debe repetirse una acción. Este ciclo cuenta con cuatro elementos clave:

  • Variable de inicio: se declara e inicializa antes de comenzar el ciclo.
  • Condición: determina hasta cuándo se ejecutará el bucle.
  • Modificador: cambia el valor de la variable en cada iteración.
  • Cuerpo del bucle: contiene las instrucciones que se repetirán.

Ejemplo de un bucle for en C#:


    for (int i = 0; i < 5; i++) 
    {
        Console.WriteLine("Iteración número: " + i);
    }

En este código:

  • Variable de inicio: int i = 0; se declara y asigna el valor inicial.
  • Condición: i < 5; el bucle continuará mientras i sea menor que 5.
  • Modificador: i++; incrementa i en cada iteración.
  • Cuerpo del bucle: imprime en pantalla el número de iteración.

Bucle while

El bucle while ejecuta repetidamente un bloque de código mientras se cumpla una condición lógica. Se usa cuando no se conoce el número exacto de iteraciones desde el inicio.

Ejemplo en C#:


    int contador = 0;
    while (contador < 5) 
    {
        Console.WriteLine("El contador es: " + contador);
        contador++;
    }

Aquí, el ciclo continuará ejecutándose mientras la variable contador sea menor que 5. En cada iteración, su valor aumenta en 1.

Bucle do while

El bucle do while es similar al while, pero con la diferencia de que el código dentro del bucle se ejecutará al menos una vez, incluso si la condición no se cumple.

Ejemplo en C#:


    int numero = 0;
    do 
    {
        Console.WriteLine("El valor de numero es: " + numero);
        numero++;
    } while (numero < 5);

En este caso, el código dentro del bloque do se ejecuta primero y luego se evalúa la condición. Si sigue siendo verdadera, el bucle se repetirá.

4️⃣ Funciones

Las funciones son bloques de código reutilizables que permiten estructurar mejor un programa, mejorar la legibilidad y evitar la repetición de código innecesario. Cada función puede recibir datos de entrada, procesarlos y devolver un resultado.

Características de las Funciones

Las funciones en programación tienen varias características clave:

  • Modularidad: Permiten dividir el código en partes más manejables.
  • Reutilización: Se pueden llamar múltiples veces sin reescribir el código.
  • Encapsulación: Mantienen su propia lógica sin afectar otras partes del programa.
  • Parámetros: Pueden recibir valores de entrada para trabajar con ellos.
  • Valor de retorno: Pueden devolver un resultado que se usará en otra parte del código.

Sintaxis Básica de una Función

En C#, una función se define utilizando el siguiente formato:


    tipoDeRetorno NombreFuncion(parámetros)
    {
        // Cuerpo de la función
        return valor; // (opcional)
    }

Ejemplo de una función en C# que suma dos números:


    int Sumar(int a, int b) 
    {
        return a + b;
    }

En este ejemplo:

  • Tipo de retorno: int indica que la función devuelve un número entero.
  • Nombre de la función: Sumar describe su propósito.
  • Parámetros: int a, int b son los valores que recibe la función.
  • Valor de retorno: return a + b; devuelve la suma de los dos números.

Funciones sin Retorno

Algunas funciones no necesitan devolver un valor, solo ejecutan una acción. Estas se definen con el tipo de retorno void.


    void Saludar(string nombre) 
    {
        Console.WriteLine("Hola, " + nombre + "!");
    }

Aquí, la función Saludar imprime un mensaje en pantalla sin devolver un valor.

Funciones con Parámetros Opcionales

En C#, se pueden definir parámetros opcionales con valores por defecto. Si el usuario no proporciona un valor, se usará el predeterminado.


    void MostrarMensaje(string mensaje = "Mensaje por defecto") 
    {
        Console.WriteLine(mensaje);
    }

Llamar a MostrarMensaje() sin argumento imprimirá "Mensaje por defecto".
